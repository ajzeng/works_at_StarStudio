可以修改main.js文件，通过弹窗接口设置弹窗行为，具体如下(可以链式调用)：

提示： 已经将弹窗对象挂在window对象的pop属性下，如果不想修改main.js， 可以打开chrome浏览器的调试工具，在console处进行下面操作。


1. pop.set_visible(flag); // flag为true显示弹窗，false关闭弹窗

2. pop.set_move(flag); // flag为true可以按住标题栏拖动，false关闭此功能

3. pop.set_resize(flag); // flag为true可以拖动右边框或下边框或右下角改变弹窗尺寸，false关闭此功能

4. pop.set_adaptive(flag); // flag为true可以让弹窗尺寸自适应内容，false关闭此功能
